# Νόημα Ζωής — 28 Ημέρες (Vite + React)

Έτοιμο project για άμεσο deploy σε Vercel / Netlify.

## Οδηγίες γρήγορα (Vercel)
1. Κάνε unzip και ανέβασε όλο το φάκελο στο νέο project στο Vercel.
2. Το Vercel θα τρέξει `npm install` και `npm run build` αυτόματα.
3. Το site θα είναι διαθέσιμο σε σύνδεσμο `https://<your-project>.vercel.app`

## Τοπικά
```bash
npm install
npm run dev
```

## Σημείωση
Το project περιλαμβάνει Tailwind και Vite. Αν θες, μπορώ να σε καθοδηγήσω βήμα-βήμα στο deploy.
